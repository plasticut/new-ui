new-ui
=====================

_startup_:
```shell
npm install -g bower grunt-cli
npm install
npm start
```

_components_:
* lodash
* jquery
* backbone
* backbone.modelbinder
* backbone-computedfields
* backbone-deep-model
* bootstrap

_features_:
* browserify modules
* style preprocessor less + autoprefixer + csscomb
* livereload
