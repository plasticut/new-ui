new-ui
=====================

_startup_:
```shell
npm install -g bower grunt-cli
npm install
npm start
```

_generate and serve api documentation_:
```shell
npm docs
```

_components_:
* lodash
* jquery
* backbone
* backbone.modelbinder
* backbone-computedfields
* backbone-deep-model
* semantic-ui

_features_:
* browserify modules
* style preprocessor stylus with nib
* livereload
* JSDoc 3 - API documentation generator http://usejsdoc.org/
